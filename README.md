This project helps educational institutions reduce manual work, improve accuracy, and generate reports efficiently.

## 📌 Project Overview

The **Automated Marksheet Generation and Evaluation System** is designed to simplify the process of entering student marks, 
calculating totals/percentages, assigning grades, and generating final marksheets.

It provides an organized platform for administrators, staff, and institutions to manage student academic data digitally.

---

## 🚀 Features

- Student record management
- Subject-wise marks entry
- Automatic total and percentage calculation
- Grade generation based on marks
- Marksheet generation
- Search student results
- Update/Delete student records
- Admin login panel
- Database integration
- Printable reports
